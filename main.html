Welcome

As of right now you can run an 8 or 16 player double or single elimniation tournament brackets. This website is free to use for any type of competition you wish to have. You don't need to create an account or have any type subscription, just create your bracket, add the players and run it.

You can enter a lock code if you wish to stop others from updating your tournament bracket or leave it open so anyone can do anything to it.

The website is still currently undergoing design and update features as I hope to make it better and easier to use.
